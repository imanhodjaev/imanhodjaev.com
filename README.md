# Frozen Flask + Github Pages Example

My static web site with [Flask][], you can fork and deploy them for free on [Github Pages][].

See the blog post [here][article].

```bash
$ pip install -r requirements.txt
# Run local server
$ python run.py
# Build static content
$ python freeze.py
```
